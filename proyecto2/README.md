# Proyecto 2 - ATM Simulator (Login)

Este repositorio contiene el proyecto 2: un simulador de cajero automático con pantalla de login.

Contiene:
- `proyecto2.html` - página principal del login
- `img/` - recursos gráficos
- Estilos compartidos cargados desde `style.css` en la carpeta padre

Instrucciones rápidas:
1. Abrir `proyecto2/menu.html` en el navegador para ver el dashboard o `proyecto2/ajustes.html` para ajustes.
2. Ver `DOCUMENTACION.md` para detalles completos, selectores y comandos para ejecutar localmente.
